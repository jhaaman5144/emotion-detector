About This App

This application uses machine learning to detect emotions in text. The model analyzes your text and classifies it as one of the following emotions:

Happy Sad Angry Neutral

How It Works

The detection system works through these steps:

  1. Your text is preprocessed (removing stopwords, special characters, etc.)
  2. The cleaned text is transformed into features using TF-IDF vectorization
  3. A trained machine learning model predicts the most likely emotion
  4. The results are visualized showing both the detected emotion and probability distribution
